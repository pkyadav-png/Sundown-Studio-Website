🎨 Sundown Studio Clone
A high-performance, visually stunning landing page recreation inspired by the Sundown Studio design. 
This project showcases advanced frontend techniques, seamless animations, and a modern aesthetic.

🌟 Highlights
. Dynamic Hover Effects: Unique image-reveal animations on list items.

. Smooth Navigation: Seamless transitions using Lenis.js.

. Modern UI: Clean layout with a focus on bold typography and video integration.

. Swiper Integration: Interactive touch-slider for the services/review section.

. Responsive for all screen szie.

🛠️ Built With
. HTML5 & CSS3 (Custom properties & Keyframes)

. JavaScript (DOM Manipulation)

. GSAP (GreenSock Animation Platform)

. Lenis.js (Smooth scrolling)

. Swiper.js (Modern touch slider)
