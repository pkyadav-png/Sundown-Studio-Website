# Sundown Studio Clone 🚀

A fully responsive and high-end animated website clone of **Sundown Studio**. This project focuses on smooth scrolling, modern UI design, and complex web animations.

## ✨ Features

* **Smooth Scrolling:** Implemented using Locomotive JS / Lenis.
* **Advanced Animations:** Created with GSAP (GreenSock Animation Platform) and ScrollTrigger.
* **Responsive Design:** Fully optimized for all screen sizes (Desktop, Tablet, and Mobile).
* **Custom Typography:** Premium fonts integrated for a sleek look.
  

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Custom properties and Flexbox/Grid)
* **JavaScript** (ES6+)
* **GSAP** (for animations)
* **Swiper.js** (for sliders)
* **Lenis.js** (for smooth scrolling)


## 🚀 How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Enjoy the animations!
